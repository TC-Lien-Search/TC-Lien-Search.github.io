# TCLienSearch.com

Invoicify app resides within the `invoicify` sub-directory. Place Invoicify app build files into this folder and commit and push the changes for them to be made live.

App uses [EmailJS](https://www.emailjs.com/) for sending emails and the [GeoCodeAPI](https://app.geocodeapi.io) for getting valid addresses on the order form. Tara will have the login info for these services.
